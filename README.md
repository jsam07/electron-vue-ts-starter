## Getting Started

### 1. Download starter and install dependencies

Clone this repository:

```
git clone git@github.com:BCIT-DDC/electron-ts-starter.git
```

Install npm dependencies:

```
cd electron-ts-starter
npm install
```

### 2. Start Desktop Application

```
npm run start
```
